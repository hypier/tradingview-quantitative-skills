# 日历事件追踪工作流

追踪财经日历中的重要市场事件，包括财报发布、分红派息、IPO上市、经济数据发布等。

## 执行步骤

### 步骤 1: 确定日历类型

根据用户需求确定日历类型：
- **economic**: 经济日历（GDP、CPI、就业数据等）
- **earnings**: 财报日历（公司财报发布日期）
- **revenue**: 分红日历（股息派息日期）
- **ipo**: IPO日历（新股上市日期）

### 步骤 2: 确定时间范围

计算查询的时间范围（Unix 时间戳）：
- 默认查询未来 7-14 天
- 用户可指定具体日期范围
- 注意：时间跨度不能超过 40 天

### 步骤 3: 确定市场范围

根据日历类型选择市场：
- 经济日历：支持多国家（america, china, japan 等）
- 财报/分红/IPO：默认 america，可指定其他市场

### 步骤 4: 调用日历工具

调用 `tradingview_get_calendar` 获取日历数据：

```
参数说明：
- type: 日历类型（必填）
  - economic: 经济事件
  - earnings: 财报发布
  - revenue: 分红派息
  - ipo: 新股上市
- from: 开始时间（必填，Unix 时间戳）
- to: 结束时间（必填，Unix 时间戳）
- market: 市场代码（可选，逗号分隔）
```

### 步骤 5: 筛选和排序事件

根据用户关注点筛选：
- 按重要性排序
- 按行业/板块筛选
- 按具体标的筛选

### 步骤 6: 生成日历报告

输出格式化的日历报告：
- 事件列表（时间、事件名称、预期值、实际值）
- 重要事件高亮
- 潜在影响分析
- 建议关注的标的

## 示例对话

**用户**: "下周有哪些重要财报？"

**执行**:
1. 计算下周的时间范围（from/to 时间戳）
2. 调用 `tradingview_get_calendar`，type="earnings"
3. 返回财报列表，标注市场关注度高的公司
4. 建议关注可能超预期的标的

---

**用户**: "查看未来两周的美国经济数据发布"

**执行**:
1. 计算未来两周的时间范围
2. 调用 `tradingview_get_calendar`，type="economic"，market="america"
3. 返回 CPI、非农、GDP 等重要数据发布时间
4. 分析数据对市场的潜在影响

---

**用户**: "查看本月有哪些新股上市"

**执行**:
1. 计算本月的时间范围
2. 调用 `tradingview_get_calendar`，type="ipo"
3. 返回新股列表，包含发行价、上市日期等
4. 分析新股的行业分布和打新建议

---

**用户**: "我持有的股票近期有分红吗？"

**执行**:
1. 获取用户持仓股票列表
2. 调用 `tradingview_get_calendar`，type="revenue"
3. 筛选用户持仓股票的分红信息
4. 返回除息日、派息日、分红金额等
