# 技术分析方法论

> 专业的技术分析评分模型和指标解读

---

## 📊 综合评分模型

### 总分构成 (100分)

```
总分 = 趋势强度(30分) + 动量指标(25分) + 形态识别(20分) + 
       支撑阻力(15分) + 市场情绪(10分)
```

### 评级标准

| 分数区间 | 评级 | 建议 |
|---------|------|------|
| 90-100 | ⭐⭐⭐⭐⭐ | 强烈推荐 |
| 80-89 | ⭐⭐⭐⭐ | 推荐 |
| 70-79 | ⭐⭐⭐ | 中性观望 |
| 60-69 | ⭐⭐ | 谨慎 |
| < 60 | ⭐ | 不推荐 |

## 1️⃣ 趋势强度分析 (30分)

### 1.1 多周期均线排列 (0-15分)

#### 完美多头排列 (15分)
```
5日均线 > 10日均线 > 20日均线 > 60日均线
```

#### 评分标准
- 5日 > 10日 > 20日 > 60日: **15分** (完美多头)
- 5日 > 10日 > 20日: **12分** (强势多头)
- 5日 > 10日: **8分** (短期多头)
- 均线粘合: **5分** (方向不明)
- 空头排列: **0分** (趋势向下)

### 1.2 价格相对位置 (0-10分)

#### 评分标准
- 价格 > 所有均线: **10分** (强势)
- 价格 > 60日均线: **8分** (中期强势)
- 价格 > 20日均线: **5分** (短期强势)
- 价格 < 所有均线: **0分** (弱势)

### 1.3 趋势持续时间 (0-5分)

#### 评分标准
- 持续 > 30天: **5分** (趋势成熟)
- 持续 20-30天: **4分** (趋势确立)
- 持续 10-20天: **3分** (趋势初期)
- 持续 < 10天: **1分** (趋势不稳)

## 2️⃣ 动量指标分析 (25分)

### 2.1 RSI 相对强弱指数 (0-10分)

#### 健康度评分
- RSI 在 40-60: **10分** (最健康)
- RSI 在 30-40 或 60-70: **8分** (健康)
- RSI 在 20-30 或 70-80: **5分** (警戒)
- RSI < 20 或 > 80: **2分** (超买超卖)

#### 交易信号
```
买入信号:
- RSI 从下向上突破 30
- RSI 底背离（价格创新低，RSI 不创新低）

卖出信号:
- RSI 从上向下跌破 70
- RSI 顶背离（价格创新高，RSI 不创新高）
```

### 2.2 MACD 指标 (0-10分)

#### 状态评分
- 金叉 + DIF > 0 + 柱状图放大: **10分** (强势买入)
- 金叉 + DIF < 0: **8分** (买入信号)
- 金叉 + 柱状图缩小: **6分** (动能减弱)
- 死叉 + DIF > 0: **3分** (警惕)
- 死叉 + DIF < 0: **0分** (卖出信号)

#### 交易信号
```
强烈买入:
- MACD 在 0 轴上方金叉
- 柱状图快速放大

买入:
- MACD 在 0 轴下方金叉
- 底背离

卖出:
- MACD 在 0 轴下方死叉
- 顶背离
```

### 2.3 成交量配合 (0-5分)

#### 量价关系评分
- 上涨 + 放量: **5分** (健康上涨)
- 上涨 + 缩量: **3分** (动能不足)
- 下跌 + 缩量: **2分** (抛压减轻)
- 下跌 + 放量: **0分** (恐慌性下跌)

#### 相对成交量
```
放量: 当日成交量 > 5日均量 × 1.5
缩量: 当日成交量 < 5日均量 × 0.7
```

## 3️⃣ 形态识别 (20分)

### 3.1 经典形态 (0-15分)

#### 反转形态
- **双底/双顶**: 置信度 > 85% → 15分
- **头肩底/头肩顶**: 置信度 > 80% → 12分
- **圆弧底/圆弧顶**: 置信度 > 75% → 10分

#### 整理形态
- **上升三角形**: 置信度 > 80% → 12分
- **旗形整理**: 置信度 > 75% → 10分
- **矩形整理**: 置信度 > 70% → 8分

### 3.2 K线形态 (0-5分)

#### 看涨形态
- **锤子线**: 出现在支撑位 → 5分
- **看涨吞没**: 成交量放大 → 5分
- **启明星**: 三根K线组合 → 5分

#### 看跌形态
- **倒锤子**: 出现在阻力位 → 0分
- **看跌吞没**: 成交量放大 → 0分
- **黄昏星**: 三根K线组合 → 0分

## 4️⃣ 支撑阻力分析 (15分)

### 4.1 距离支撑位 (0-8分)

#### 评分标准
```
距离支撑位 < 3%: 8分 (安全边际高)
距离支撑位 3-5%: 6分 (较安全)
距离支撑位 5-8%: 4分 (一般)
距离支撑位 > 8%: 2分 (风险较大)
```

### 4.2 突破阻力位 (0-7分)

#### 评分标准
```
有效突破阻力位 + 放量: 7分 (强势)
突破阻力位但缩量: 4分 (待确认)
接近阻力位未突破: 2分 (观望)
远离阻力位: 0分 (空间有限)
```

### 4.3 支撑阻力计算方法

#### 1. 历史高低点法
```javascript
// 找出近60日的重要高低点
function findSupportResistance(priceHistory) {
  const highs = findLocalHighs(priceHistory);
  const lows = findLocalLows(priceHistory);
  
  // 支撑位：近期重要低点
  const support = Math.max(...lows.slice(-3));
  
  // 阻力位：近期重要高点
  const resistance = Math.min(...highs.slice(-3));
  
  return { support, resistance };
}
```

#### 2. 均线支撑法
```
短期支撑: 20日均线
中期支撑: 60日均线
长期支撑: 120日均线
```

#### 3. 黄金分割法
```
从高点回调:
- 0.382 回调位 (浅回调)
- 0.5 回调位 (中度回调)
- 0.618 回调位 (深度回调)

从低点反弹:
- 1.382 目标位
- 1.618 目标位
- 2.0 目标位
```

## 5️⃣ 市场情绪分析 (10分)

### 5.1 相对强弱 (0-5分)

#### 评分标准
```
跑赢大盘 > 10%: 5分 (超强)
跑赢大盘 5-10%: 4分 (强势)
跑赢大盘 0-5%: 3分 (略强)
跑输大盘 0-5%: 2分 (略弱)
跑输大盘 > 5%: 0分 (弱势)
```

### 5.2 板块联动 (0-5分)

#### 评分标准
```
板块龙头 + 板块强势: 5分
板块跟随 + 板块强势: 4分
板块龙头 + 板块一般: 3分
板块跟随 + 板块一般: 2分
板块落后: 0分
```

## 🎯 使用建议

### 1. 综合判断
- 不要只看单一指标
- 结合多个维度综合评分
- 关注评分变化趋势

### 2. 动态调整
- 每日更新评分
- 跟踪评分变化
- 及时调整策略

### 3. 风险控制
- 高分股票也要设止损
- 低分股票可能超跌反弹
- 严格执行风险管理

### 4. 市场环境
- 牛市: 70分以上可考虑
- 震荡市: 80分以上
- 熊市: 90分以上或超跌反弹
